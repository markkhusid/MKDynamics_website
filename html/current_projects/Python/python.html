Simple Program to Draw a Circle in Python

The following script makes use of the math and turtle modules available in Python to draw a circle on the screen.

drawcircle.py
Run the code by typing:

$ python drawcircle.py

at the command prompt.

After typing that command, a window opens up and an animated turtle draws a circle.

Program to Draw a Spirograph in Python

The following script makes use of the modules available in Python to draw a circle on the screen.

spiro.py
Run the code by typing:

$ python spiro.py

at the command prompt.

After typing that command, a window opens up and an animated set of spirographs are drawn.

Program to simulate Conway's Game of Life

The following script makes use of the matplotlib module available in Python to animate a function and save the animation to an MP4 movie file.

conway.py
Run the code by typing:

$ python conway.py --mov-file filename.mp4

at the command prompt, where filename is the name of the movie file.

After typing that command, a window opens up and an animated Conway's Game of Life displays.
