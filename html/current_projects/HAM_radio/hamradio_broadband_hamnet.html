Introduction

In this section I will demonstrate a broadband hamnet radio link between two laptops. The link is composed of two Linksys WRT54GL routers that have had their firmware reflashed with the Broadband Hamnet firmware available here.

Setting up Node 1

After the WRT54GL routers were reflashed with the Broadband Hamnet firmware, I logged into Node 1 by entering http://localnode:8080 in the URL entry bar. This brought up the main screen which displays signal telemetry as well as a button for setup.

WRT54GL Router for Node 1

Here we show Node 1's WRT54GL router.

Setting up Node 2

Node 2's main screen is brougt up by accessing localnode:8080 on the second laptop.

Video of Accessing a Website on Node 1's Laptop from Node 2's Laptop

Video of laptop attached to Node 2 accessing a website residing on the laptop attached to Node 1

In this video I demonstrate accessing a website that resides on the laptop attached to Node 1 from a laptop attached to Node 2. The website is accessed by entering ac2lh-node1 into the url bar of the browser opened on Node 2's laptop. The request goes out over the broadband hamnet and Node 1 is configured to forward port 80 requests to the laptop attached to Node 1. Node 1 has an Apache webserver started and responds to this request.
