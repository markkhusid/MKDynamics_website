Introduction

In this section we will be writing shellcode for the ARM platform. We will try to mirror the shellcoding for x86 section as closely as possible. In other words, every time we make an accomplishment for the x86 platform, we will strive to repeat that accomplishment for the ARM platform.

The shellcode we will be writing here will set the setruid to that of root and spawn a shell with root level privileges.

Project code is contained in my github.

The program shellcode.s

The screenshot below displays the contents of the program shellcode.s. We need to assemble, link, run and instruction step this program in GDB.

Since our computer is an AMD64 platform, we will execute the assembly code on a Raspberry Pi 2, which is an ARM platform. We will connect to a running RPi2 via the VNC viewer.

The program is assembled with:

$ as -ggdb3 shellcode.s -o shellcode.o
