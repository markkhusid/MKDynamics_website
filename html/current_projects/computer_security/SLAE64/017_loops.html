Introduction

In this section we will be following along with and completing course material from the Pentester Academy's SecurityTube Linux Assembly language Expert 64 Bit course. In this section we will be studying loops in 64 Bit assembly language. It is to be coded, assembled, linked and run. Execution is then to be monitored via GDB. We will use the GEF add-on for GDB to enhance the visual presentation from GDD, as well as viewing the program flow in Radare2.

Project code is contained in my github.

The program Loops64.nasm

The program is assembled with:

$ nasm -felf64 Loops64.nasm -o Loops64.o
And linked with:
$ ld Loops64.o -o Loops64
