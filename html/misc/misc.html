Whonix is a system that is composed of two downloadable virtual machines. One is a gateway, the other is a workstation. They separate the internet access between them, so that the workstation sends traffic through the gateway, which forwards the traffic through TOR. The separation and use of virtual machines adds greater anonymity because certain types of identification attacks track a user based on machine configuration. Instructions for installing:
1) Unpack everything into directory of your choice.
2) In windows, start cmd.exe and type:
convert gxrom-r ROM.48G
You should get a operation completed successfully message.
3) Start EMU48.EXE and navigate the menus to choose a suitable KML script (skin) for the calculator.
4) For Linux users, perform step one as stated.
4a) Start WINE Explorer and navigate to cmd.exe in the directory /windows/command.
4b) Perform step two and return here.
5) You can start the emulator by typing wine emu48.exe or create a shortcut.
6) To create a pleasing icon image, convert one of the high res images in the Emu48 directory from .bmp to .png, which Linux uses.
7) Set this .png file as the icon.
8) Add the newly created shortcut to your taskbar if desired, for quick access to your favorite calculator!
